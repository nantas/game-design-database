# Designing The Core Gameplay Loop: A Beginner’s Guide

## 内容摘要
- 文章将 gameplay loop 定义为玩家会反复执行的动作序列，它构成了持续游玩的主流程。
- 循环可以非常简单，例如识别障碍并跳跃躲避；也可以非常复杂，例如组织团队、击败敌人并分配战利品。
- core gameplay loop 是其他外层循环的基础，外围系统应围绕这条核心行为链提供目标、反馈与延展。
