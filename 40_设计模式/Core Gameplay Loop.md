# Core Gameplay Loop

## 定义

- 核心玩法循环是玩家在一次完整体验中会反复执行的最小行为闭环，通常由行动、反馈、结果与再次行动的动机组成。
- 它是其他成长、任务、经济或社交循环赖以建立的基础。

## 适用场景

- 新项目立项时需要明确“玩家反复在做什么”。
- 设计外围成长、任务或奖励系统前，需要先确认核心乐趣是否稳定。
- 诊断留存问题时，需要判断玩家是否在第一轮循环内就感受到目标与反馈。

## 设计价值

- 让团队围绕同一个核心行为统一设计讨论口径。
- 帮助外层系统围绕核心操作提供更一致的目标、反馈与奖励。
- 降低“外围系统很多，但核心体验不成立”的设计风险。

## 常见变体

- 战斗 → 反馈 → 奖励 → 再战斗
- 采集 → 建造/升级 → 解锁新采集目标
- 关卡挑战 → 结算奖励 → 强化角色/装备 → 再挑战
