# Game Design Skills

## 来源定位

- 面向游戏设计从业者与学习者的长文知识站点，覆盖系统设计、关卡、经济、叙事与方法论。

## 可信度判断

- 适合作为高价值二手知识来源，尤其适合提炼术语定义、设计框架与入门方法。
- 站内也包含职业发展与求职类内容，不应把这部分文章直接当作机制结论或市场事实。

## 常见内容类型

- 游戏设计概念解析
- 系统、经济、关卡与叙事专题长文
- 职业路径、作品集与面试建议

## 导入注意事项

- 页面常见 WordPress / Elementor 正文容器，不能只依赖 `Deconstructor of Fun` 样例中的 `.sqs-html-content`。
- 页面里可能同时存在发布时间与更新时间；导入时应优先保留发布时间，避免把修订时间当成首发时间。
- 标题常带 `A Beginner’s Guide` 一类教学后缀，模式卡主词条需要人工归并。
