# STS2 Pattern Candidate Workbench

## 使用规则
- 每轮最多 3 条候选。
- 每条必须回链核心体验锚点。
- 每条必须标注可复刻性（当前游戏内）。
- 每条必须标注证据强度（可确认 / 低风险 / 待验证）。
- 子模式默认需要同一游戏内至少两个不同机制/内容条目共同指向；单条强证据需用户确认。

## 候选池

### 候选 A：分层循环可学习性模式（母模式）
- 核心体验锚点：高价值决策压力；失败-学习-改进循环；从首局到高峰体验的连续反馈传导。
- 机制/内容条目：
  - 单回合出牌顺序决策与即时反馈。
  - 单场战斗资源规划与结算反馈。
  - 单层路线与风险管理对整局构筑窗口的传导。
  - 单局构筑目标对局部决策的反向约束。
- 可复刻性判断（当前游戏内）：高。可通过新增角色资源机制、关键词、遭遇约束与奖励结构继续扩产。
- 证据映射：Steam 商店页 + Polygon 评测 + IGN 评测 + 核心体验页当前结论。
- 证据强度：可确认（母模式层）；子模式仍待后续拆解与补证。
- 用户裁决：纳入模式。
- 裁决备注：不存在无意义选择；通过循环分层让玩家在可控代价下学习正确策略，以支撑全程强决策与强反馈体验。

### 候选 B：跨 Act 延迟回报决策
- 核心体验锚点：长周期规划与延迟回报。
- 机制/内容条目：跨 Act 任务卡（如 Act 1 地图 -> Act 2 宝藏）。
- 可复刻性判断（当前游戏内）：中。
- 证据映射：IGN 评测中的跨 Act 事件描述。
- 证据强度：低风险（证据样本偏单点）。
- 用户裁决：暂不纳入。
- 裁决备注：当前按反模式处理，避免证据偏移；路线策略结论等待其他证据后再归纳。

### 候选 C：Event Log 复盘增强
- 核心体验锚点：失败后复盘效率。
- 机制/内容条目：`Event Log` 模块存在。
- 可复刻性判断（当前游戏内）：中。
- 证据映射：Wiki 结构层模块存在。
- 证据强度：待验证（缺少玩家使用行为证据）。
- 用户裁决：跳过。
- 裁决备注：不是明显模式，暂不进入模式清单。

## 下一轮入口
- 继续保持每轮 3 条候选。
- 仅当出现“同类且提供更细机制/内容指导”的条目时，才考虑新增子模式并回链本母模式。
- 在形成至少 3 个不同游戏卡样例前，不升级为跨游戏通用模板。

## 第 2 轮候选

### 候选 D：异质资源轴模式
- 核心体验锚点：资源调度决定高价值判断；局部资源投入会改变整局构筑收束方向。
- 机制/内容条目：
  - `Regent` 的 `star` 第二资源。
  - `Necrobinder` 的骷髅手与 `Doom` 双向消耗关系。
  - 新卡牌修饰符对能量与使用节奏的改写。
- 可复刻性判断（当前游戏内）：高。可通过新增角色资源轴、新关键词资源与新型结算资源继续扩产。
- 证据映射：
  - IGN：`Regent` 的 `star` 第二资源。
  - IGN：`Necrobinder` 的骷髅手与 `Doom`。
  - IGN：新卡牌修饰符带来更多构筑灵活性。
- 证据强度：低风险。
- 当前归纳：该模式应优先被视为资源设计主模式，而不是“应对异质资源的学习模式”。
- 用户裁决：纳入模式。

### 候选 E：基础强度模板模式
- 核心体验锚点：玩家需要先建立对卡牌基础价值的稳定判断，才能在构筑过程中做出高质量取舍。
- 机制/内容条目：
  - “经典卡牌基于能量分配策略被重新语境化”这一描述，反向证明存在可被重新定价的基础模板。
  - 角色、卡牌、遗物作为可组合内容池，要求玩家先识别共通的基础强度基线。
- 可复刻性判断（当前游戏内）：中高。可通过持续新增卡池时保持若干标准模板来扩产。
- 证据映射：
  - Polygon：经典卡牌在新机制下被重新语境化。
  - 游戏内容页：角色、卡牌、遗物、敌人与地图节点共同组成可组合内容池。
- 证据强度：低风险。
- 当前归纳：这是“重定价模式”的前置模式，没有基础模板，就不存在稳定的再评价。
- 用户裁决：纳入模式。

### 候选 F：特定语境或策略下重定价模式
- 核心体验锚点：正确判断的快感来自“同一张牌/同一类效果”在不同资源语境下价值发生明显变化。
- 机制/内容条目：
  - 新卡牌和遗物“扭转玩家预期”。
  - 经典卡牌因能量分配策略与新触发条件被重新语境化。
  - 新修饰符改变卡牌的复用、衰竭与首次出牌收益。
- 可复刻性判断（当前游戏内）：高。可通过更多环境触发器、资源门槛和修饰符继续扩产。
- 证据映射：
  - Polygon：新卡牌和遗物扭转玩家预期。
  - Polygon：经典卡牌在新机制下被重新语境化。
  - IGN：修饰符改变防御牌、额外能量与复用方式。
- 证据强度：低风险。
- 当前归纳：该模式与“基础强度模板模式”成对出现，前者负责建立基线，后者负责在具体语境中重估价值。
- 用户裁决：纳入模式。

### 候选 G：资源转换模型模式
- 核心体验锚点：策略深度不仅来自资源多样性，也来自资源之间可交换、可延期、可损耗的转换关系。
- 机制/内容条目：
  - `Doom` 对敌我血量的双向消耗，可视为生命/效果之间的转换压力。
  - 修饰符对“使用一次”“衰竭后增益”“首次出牌给能量”的设计，可视为时机与收益的转换器。
  - 单场战斗资源规划强调“现在交还是留给下一轮”，说明资源转换不只存在于卡面，也存在于战斗节奏。
- 可复刻性判断（当前游戏内）：高。可通过更多资源交换规则、延迟收益与风险对冲机制扩产。
- 证据映射：
  - IGN：`Doom` 双向消耗。
  - IGN：修饰符对复用、衰竭、首次出牌收益的影响。
  - 核心体验页：单场战斗资源规划与投入/保留决策。
- 证据强度：低风险。
- 当前归纳：该模式更偏机制设计指导，可作为异质资源轴模式下的细化方向。
- 用户裁决：纳入模式。
